File and folder structure

The following is the file and folder structure for the eleventy State Web Template package. 

StateTemplate/
├── _site/
└── pages/
   ├── index.html
   └── etc...
└── src/
   └── _includes/
        ├── global-footer.njk
        ├── etc...
        └── layouts/
        ├── base.njc
        └── etc...
   └── css/
   └── images/
   └── js/
   └── root/
├── eleventy.js
├── package.json
└── rollup.config.js
  • _site - This folder contains website's output files.
  • pages - Contains website's input pages that a developer or e content editor can edit.
  • src - The “src” folder contains includes, modules, custom CSS and JavaScript files.
  • _includes - This folder contains the common template components used on various pages in the State Web Template (for example, header, footers, navigation, etc.).
  • css - Can contain custom css files that can be referenced in the front mater of the specific pages or in the head area of the head-js-css.njk include
  • images - This folder contains all the images used by the State Template.
  • js - Contains custom.js file that potentially can be used for any custom JavaScript code
  • root - Contains website's essential files such as sitemap, web.config, favicon, robots, etc.
