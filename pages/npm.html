State Web Template NPM package

This eleventy project is using State Web Template NPM package that is exported from the node_modules into /state-template/ directory when an user is executing the npm build or npm start commands.

How it works

To load NPM State Web Template package we have added following code into our eleventy.js file. 

module.exports = function(eleventyConfig) {
  // Copy state template core files to site output
  eleventyConfig.addPassthroughCopy({
    "node_modules/@cagovweb/state-template/dist": "state-template"
  });
};

Once State Web Template package is loaded into our project we can make references to the State Web Template's core CSS and JS files in our head-js-css.njk inclide: 

<head>
  ...
  <!-- State Web Template core files -->
  <link rel="stylesheet" href="/state-template/css/cagov.core.min.css">
  <link rel="stylesheet" href="/state-template/css/colortheme-oceanside.css">

  <script src="/state-template/js/cagov.core.min.js"></script>

 </head>
