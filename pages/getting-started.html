Get started

If you are new to the California Web State Template please follow instroctions below that will help you started.

Note: if you are a returning State Web Template user please check Template Updates page for the latest updates notes and updating to the latest version instructions.

Download and setup

  1. Clone or download the State Web Template from GitHub
    p>To clone the State Web Template go to our GitHub repository to copy template's repository URL.


    Or download latest State Web Template release ZIP file directly from GitHub releases page.
  2. Set up your page

    You can use State Template existing web pages to start. However, if your website doesn't use NPM or eleventy site generator use clean starter template code below.

    Clean HTML template source code

      <!doctype html>
  <html lang="en">
    <head>
      <!-- Required meta tags -->
      <meta charset="utf-8">
      <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
  
      <!-- State Template CSS -->
      <link rel="stylesheet" href="/css/cagov.core.css">
      <link rel="stylesheet" href="/css/colortheme-oceanside.css">

    </head>
    <body>
      <h1>California State Webpage</h1>
  
      <!-- State Template JavaScript -->
      <script src="/js/cagov.core.js" ></script>
    </body>
  </html>

  3. Using CSS and JS

    CSS

    State Web Templagte core css files are generated from SASS files that can be found in /src/scss/ folder. Multiple SASS files are compiled during the build process into core output CSS files that are generated in /_site/css/ folder. A State Web Template webpage requires these two CSS files:

       <link rel="stylesheet" href="/css/cagov.core.css">
   <link rel="stylesheet" href="/css/colortheme-oceanside.css">

    If your website doesn't use a site generator you can simply copy-paste the stylesheet links into your <head> before all other stylesheets to load our CSS. please note that you can replace colortheme-oceanside.css with color scheme file of your choice.

    JavaScript

    A State Web Template webpage requires cagov.core.js JavaScript file to function. That file is generated by rollup.js processor from multiple JavaScript files that can be found in /src/js/ folder. If your website doesn't use a site generator place the following near the end of your pages, right before the closing </body> tag.

    Use CDN for better performance

    Load your website faster by using State Web Template CDN. You will only need to update CSS and JavaScript CDN url links (no local CSS, JS and font files are needed).

    <link href="https://california.azureedge.net/cdt/statetemplate/6.1.1/css/cagov.core.css" rel="stylesheet">
 <link href="https://california.azureedge.net/cdt/statetemplate/6.1.1/css/colortheme-oceanside.css" rel="stylesheet">
    <script src="https://california.azureedge.net/cdt/statetemplate/6.1.1/js/cagov.core.js"></script>
  4. Using includes and page layout options

    Includes

    If you are using eleventy sie generator you can build your page by using the State Web Template includes. Several common components (such as page headers, footers, navigation, etc.) are already included in pages using njk includes.

    Note: Path to the includes will be relative to the directory of your page.

    You can also add any other html or njk includes to your page like this:

    ❴% include "../../src/_includes/modules/header-primary-banner.html" %❵

    Note: Path to the module include will be relative to the directory of your page.

    Layouts

    State Web Template eleventy page generation codebase uses several layout page options:

    landing.njk (full width page)

    page.njk (standard page)

    navicons-landing.njk (full width that has main navigation with icons)

    navicons.njk (has main navigation with icons)

    search.njk (search results page)

    components.njk or patterns.njk (page with dynamic left side navigation)

    To use use one of those option you need to specify layout key in your front matter data at the beginnig of your page. 

    ---
title: Your page title
layout: page.njk
---

    Note: You can inspect page templates source code in /src/_includes/layouts/ folder or learn more about eleventy layouts on layouts page.

  5. Running State Web Template locally

    Install Node.js

    Make sure your local machine has Node installed. To see if Node is installed, open the Windows Command Prompt, Powershell or a similar command line tool, and type:

    node -v

    This should print the version number. Othervise you can download it from nodejs.org

    Using npm to start the website locally

    Once Node is installed, and once you are in your State Web Template local directory run following npm commands to build and run the website locally:

    npm install
    This command will install all the nessesary npm packages to your local website directory. 
    npm run build
    This command will build and generate website's pages, css and js files. 
    npm run start
    This command will run the website in your http://localhost:8080/ and watch for any updates you are making to webpages or scss and js files.

    Output

    All the web content is generated into /_site/ folder, which is website's final output folder.
